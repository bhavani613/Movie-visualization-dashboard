Movie Data Visualization Dashboard

Description

This project analyzes a movie dataset using data visualization techniques to understand trends in genres, languages, ratings, and release years. The dashboard presents insights through interactive charts and graphs that help explore movie distribution and popularity.

Features

- Movies categorized by genre
- Language distribution of movies
- Year-wise movie release trends
- Movie ratings comparison
- Interactive filters for genre, language, and year

Tools Used

- Power BI / Microsoft Excel
- CSV dataset

Dataset Columns

The dataset contains the following fields:

- Title
- Genre
- Language
- Release Year
- Rating

Dashboard Visualizations

The dashboard includes the following charts:

1. Movies by Genre – Bar chart showing the number of movies in each genre.
2. Movies by Language – Pie chart displaying language distribution.
3. Movies Released Over Time – Line chart showing movie releases by year.
4. Movie Ratings Comparison – Column chart comparing ratings.
5. Interactive Filters – Slicers for genre, language, and year.

How to Use

1. Open Power BI Desktop or Excel.
2. Import the movie dataset (CSV file).
3. Create charts using the dataset fields.
4. Add filters to explore movie data interactively.

Learning Outcomes

- Data visualization techniques
- Understanding movie data trends
- Creating interactive dashboards
- Basic data analysis using charts and filters

Project Goal

The goal of this project is to transform raw movie data into meaningful insights using visual analytics tools.
